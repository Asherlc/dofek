Stripe Subscription Gate

Server-owned entitlement, Stripe-owned billing, unrestricted ingestion.

Proposed Data Flow
Clients

Web + iOS

Ask server for data, checkout URL, portal URL, and subscription status.

Server

Billing + Gate

Stores entitlement state and clamps every read query for unpaid users.

Stripe

Checkout + Portal

Collects payment, manages cancellation, sends subscription webhooks.

Paid users

Full data access when Stripe subscription is active, trialing, or an internal paid grant exists.

Unpaid users

Only reads data dated from signup day through signup day + 6. Sync/import still stores all raw history.
